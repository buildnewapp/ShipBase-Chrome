分析当前项目，实现如下功能：
支持i18n+shadcn组件库
点击 popup 弹框，显示登录界面
如果当前用户没有登录，显示登录按钮+多语言下拉切换功能，登记等于按钮登录（参考授权client登录）；多语言下拉切换语言，默认支持 英语+中文，切换后缓存保留本地；
如果用户已经登录，显示用户信息+多语言切换功能；
授权client登录
打开网页，https://freesoragenerator.com/auth/signin?client_id=abc&redirect_uri=chrome, client_id客服端随机生成；打开网页后 3s 一次轮询接口get https://freesoragenerator.com/api/auth/client?client_id=abc，如果网页端用户登录成功，会返回 token，接口返回示例：
```
{
    "code": 0,
    "message": "ok",
    "data": {
        "client_id": "abc",
        "access_token": "xxxxx",
        "token_type": "Bearer"
    }
}
```
